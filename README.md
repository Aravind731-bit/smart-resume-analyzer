# 📄 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python** and **Streamlit** that helps job seekers evaluate their resumes, improve ATS compatibility, identify missing skills, and receive personalized suggestions for career growth.

---

## 🚀 Features

- 📂 Upload Resume in PDF format
- 📄 Resume Text Extraction
- 📊 ATS Resume Score
- 🎯 Role Prediction
- 🧠 Skill Extraction
- 📈 Skill Gap Analysis
- 📝 Resume Improvement Suggestions
- 📑 Download Resume Analysis Report (PDF)
- 💻 Interactive Streamlit Web Interface

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Aravind731-bit/smart-resume-analyzer.git
```

Go to the project folder

```bash
cd smart-resume-analyzer
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
streamlit run app.py
```

The application will open automatically in your browser.

---

## 📊 How It Works

1. Upload a Resume (PDF)
2. Resume text is extracted.
3. Skills are identified.
4. ATS score is calculated.
5. Predicted job role is displayed.
6. Missing skills are highlighted.
7. Suggestions are generated.
8. Download the analysis report.
